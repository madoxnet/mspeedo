### ADNS-3080 Pinout
|Pin|Function|Connect To|
|---|---|---|
|GND|GND|GND|
|5V|Power In|3V|
|3V|???|N/C|
|NCS|/Chip Select|GPIO4|
|MISO|MISO|GPIO12|
|MOSI|MOSI|GPIO13|
|SCLK|SCLK|GPIO14|
|RST|Reset|GPIO2|
|NPD|/Power Down|N/C|
|LED|LED control output|N/C|
